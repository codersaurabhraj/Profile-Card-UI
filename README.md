# 👤 Profile Card UI

A modern and responsive **Profile Card UI** built using HTML, CSS.
This project showcases a clean design with smooth layout and social media integration.

## ✨ Features

* 🎨 Modern and aesthetic UI design
* 📱 Fully responsive profile card
* 🖼️ Profile image with circular layout
* 🔗 Social media icons integration
* 🎯 Clean typography and spacing
* ⚡ Lightweight and fast

## 🛠️ Tech Stack

* HTML5
* CSS3
